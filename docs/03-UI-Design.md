# 代码批判 - UI/UX 设计文档

## 1. 设计原则

### 1.1 视觉风格
- **风格**：现代、简洁、专业
- **主色调**：深蓝色 (#1a365d) - 代表专业、可信赖
- **辅助色**：青色 (#00d4aa) - 代表代码、技术
- **背景色**：浅灰 (#f7fafc) / 纯白 (#ffffff)
- **强调色**：橙色 (#ed8936) - 用于警告、评分

### 1.2 设计语言
- 圆角设计（8px-16px）
- 柔和阴影
- 充足的留白
- 清晰的层次结构
- 响应式布局

## 2. 页面结构

## 4. 组件设计规范

### 4.1 按钮规范

| 类型 | 背景色 | 文字色 | 圆角 | 高度 | 使用场景 |
|------|--------|--------|------|------|----------|
| Primary | #1a365d | #ffffff | 8px | 40px | 主要操作 |
| Secondary | transparent | #1a365d | 8px | 40px | 次要操作 |
| Danger | #e53e3e | #ffffff | 8px | 40px | 删除/危险 |
| Ghost | transparent | #4a5568 | 8px | 32px | 文字按钮 |

### 4.2 卡片规范
```
背景: #ffffff
圆角: 12px
阴影: 0 2px 8px rgba(0,0,0,0.08)
Hover 阴影: 0 4px 16px rgba(0,0,0,0.12)
内边距: 24px
```

### 4.3 颜色系统

```css
/* 主色调 */
--color-primary: #1a365d;        /* 深蓝 */
--color-primary-light: #2c5282;  /* 浅蓝 */

/* 功能色 */
--color-success: #48bb78;        /* 绿色 */
--color-warning: #ed8936;        /* 橙色 */
--color-error: #e53e3e;          /* 红色 */
```

### 4.4 字体规范

## 5. 响应式设计

### 5.1 断点定义
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

### 5.2 移动端适配
- 导航变为汉堡菜单
- 卡片单列布局
- 代码对比视图改为上下排列
- 按钮全宽显示
- 字体大小适当缩小

### 5.3 触摸优化
- 按钮最小点击区域 44px
- 增加元素间距
- 支持手势操作（左滑删除等）

## 6. 交互动效

### 6.1 页面过渡
- 页面切换：淡入淡出，300ms
- 路由切换：滑动效果

### 6.2 微交互
- 按钮 Hover：背景色变化，200ms
- 卡片 Hover：上浮 4px，阴影加深
- 加载状态：旋转动画
- 成功提示：勾选动画

### 6.3 加载状态
- 骨架屏：用于列表加载
- 进度条：用于文件上传、分析进度
- 旋转器：用于等待状态
